To Do Task Manager - TDTM

This Python project is a security application that combines features such as text-to-speech, camera activation on incorrect password attempts, and task management.

It utilizes libraries like `pyttsx3` for text-to-speech, `cv2` for camera access, and `tkinter` for GUI.

Upon execution, it initializes setups, creates a password-protected interface, and allows users to manage tasks through functions like adding, removing, marking as completed, and clearing tasks.

Additionally, it includes functionality to handle PDF encryption and decryption using libraries like `PyPDF2` and `fitz`.
